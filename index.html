📖 تعليمات إنشاء البوت والقناة واستخراج API
1️⃣ إنشاء بوت تيليجرام:
• افتح تيليجرام وابحث عن @BotFather
• أرسل /newbot واتبع التعليمات
• اختر اسمًا للبوت ثم اسم مستخدم ينتهي بـ bot
• ستحصل على Token يشبه: 123456:ABCdefGH...

2️⃣ إنشاء قناة تيليجرام:
• أنشئ قناة جديدة (خاصة أو عامة)
• اختر اسم مستخدم للقناة مثل @MyAppChannel

3️⃣ إضافة البوت كمسؤول للقناة:
• اذهب للقناة → الإعدادات → المسؤولون
• أضف البوت كمسؤول مع صلاحيات "نشر الرسائل" و"تحرير الرسائل"

4️⃣ الحصول على Chat ID:
• للقنوات العامة: استخدم @username مباشرة
• للقنوات الخاصة: أرسل رسالة للقناة ثم زر:
https://api.telegram.org/bot[TOKEN]/getUpdates
• ابحث عن "chat":{"id":-100xxx...}
